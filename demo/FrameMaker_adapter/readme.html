1. Running the FrameMaker adapter for the DITA Open Toolkit

This tutorial demonstrates how to generate sample documents in the DITA Open Toolkit using Ant and view the ouptut in Adobe FrameMaker.

The following tasks require the DITA Open Toolkit 1.2 or later and Adobe FrameMaker 7.0 or later. (If you are not using FrameMaker 7.2, you need to edit the value of the FrameMakerExecutable property in integrator.xml.)

Before you generate output, extract the contents of the FrameMaker_adapter.zip archive to C:\DITA-0T\demo. You must also install the adapter plug-in and add the supplied DITA-OT (Styles) application to your FrameMaker installation. For information about working with a structured applications file, refer to the Structure Application Developer's Guide that is distributed with FrameMaker.

1.1. To install the adapter plug-in

  1. Open a command window and change the path to the DITA-OT folder.
  2. On the command line, type ant -f integrator.xml and press Enter.

1.2. To add the DITA-OT application to your FrameMaker installation

  1. Open FrameMaker_adapter\structapps.fm. Select and copy the DITA-OT (Styles) application.
  2. Open structapps.fm in your FrameMaker Structure folder (for example, C:\Program Files\Adobe\FrameMaker7.2\Structure) and paste the DITA-OT (Styles) application. Edit the pathnames in the application as necessary.
  3. Select File>Structure Tools>Read Application Definitions.

1.3. To generate sample FrameMaker output

  1. On the command line, type ant -Dargs.input="samples\sequence.ditamap" dita2fmxml and press Enter. Output is copied to C:\DITA-OT\out and FrameMaker is started.
  2. In the Use Structured Application dialog, select the DITA-OT (Styles) application and press Continue. FrameMaker opens the output using the DITA-OT (Styles) application.
